🍞 Bakery Website
A modern, responsive, and interactive Bakery Landing Page built using HTML, CSS, Bootstrap, and JavaScript.
This project showcases a professional business website with smooth animations, interactive UI, and responsive design.

✨ Features
📱 Fully Responsive (Mobile – Tablet – Desktop)
🔗 Smooth Scroll Navigation
🎬 Scroll Reveal Animations
🖼️ Image Popup (Lightbox Gallery)
⭐ Testimonials Section
📩 Contact Form UI + Validation
⬆️ Scroll To Top Button
🌙 Dark Mode Toggle
🎨 Modern UI/UX & Hover Effects
⚡ Clean & Organized Code Structure

🎯 Project Purpose
This project was built to:
Practice real-world Front-End development
Apply Bootstrap layout system
Implement JavaScript DOM interactions
Create a portfolio-ready business website
Improve UI/UX and responsive design skills

🛠️ Built With
Technology	Usage
HTML5	Structure
CSS3	Styling & Animations
Bootstrap 5	Layout & Responsiveness
JavaScript (Vanilla JS)	Interactivity
